## Social site.
A simple blog post site api, with the following.
1. Login and registration
2. fetch all users.
3. fetch single user.
4. fetch all posts.
5. fetch single post.
6. like a post
7. unlike a post
8. follow other user
9. unfollow other users
10. comment on post



## built with 
* Javascript (Express with ejs and css)
* mongodb

## running and installing project
* run `npm install`
* run `npm run start`
